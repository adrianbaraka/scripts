# monitor

This utility provides a simple way to add and configure a **virtual display** on systems using Intel graphics. It is particularly useful for extending your desktop to a headless display or for use with remote desktop software.


## Usage

```bash
# Run the script to setup or start the virtual display
./monitor.sh
```

> **Note:** If this is your first time running the script, you will be prompted to reboot your system to apply the Xorg driver changes.
